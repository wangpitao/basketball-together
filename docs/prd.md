你是一名资深的微信小程序开发工程师。  
我正在开发一个微信小程序，功能是「篮球约球」，解决用户打篮球找不到人的问题。  
请帮我生成一个最小可行产品（MVP）的代码框架，基于微信小程序 + 云开发（云数据库、云函数），要求如下：  

## 功能需求
1. **首页（地图模式）**
   - 集成腾讯地图 SDK。
   - 能显示用户当前位置。
   - 在地图上展示附近篮球场（用假数据初始化）。

2. **场馆详情页**
   - 展示场馆信息（名称、地址、开放时间、是否收费）。
   - 有一个「约球留言板」，类似聊天室，可以发布约球信息。

3. **发布约球**
   - 用户可选择时间 + 人数 + 备注。
   - 点击「发布」后，存储到云数据库。
   - 场馆页自动刷新，显示最新的约球信息。

4. **加入约球**
   - 用户可以点击「加入」，报名人数实时更新。

5. **提醒功能**
   - 使用订阅消息，在约球人数满足后，通知相关用户。

## 技术要求
- 使用微信小程序原生框架。
- 数据存储使用微信云开发数据库。
- 聊天/留言板部分可以先用轮询或简单列表刷新实现（不需要真正的即时通讯）。
- 代码中请包含：
  - `app.js`, `app.json`, `app.wxss` 基本配置。
  - 页面目录结构：`pages/index` (首页地图)，`pages/venue` (场馆详情)，`pages/create` (发布约球)。
  - 云函数目录结构示例：`cloudfunctions/addGame`, `cloudfunctions/joinGame`, `cloudfunctions/getVenueList`。
- 给出数据库设计示例（如集合名称、字段说明）。

## 输出要求
请生成完整的项目文件结构，并给出关键文件的示例代码（比如 index 页面地图展示，venue 页面留言板，发布约球的表单），以便我可以直接在 Cursor 里运行和扩展。
