你是一名前端开发和产品设计师。  
请帮我生成一个「篮球约球小程序」的 **HTML 原型图**，要求如下：

## 通用要求
- 页面适配 iPhone（使用 `meta viewport`）。
- 使用 HTML + CSS，尽量简洁，不依赖框架。
- 布局基于移动端（窄屏宽度，最大宽度 430px）。
- 用方框/占位符表示组件，不需要实现业务逻辑。

## 页面需求
1. **首页（地图模式）**
   - 顶部搜索栏（输入球馆名字）
   - 中间地图区域（用灰色矩形代替）
   - 底部 tabbar（首页 / 消息 / 我的）

2. **场馆详情页**
   - 场馆信息卡片（名称、地址、开放时间）
   - 留言板区域（用列表占位）
   - 底部按钮：「发布约球」

3. **发布约球页**
   - 表单：
     - 日期和时间选择器（占位符框）
     - 人数选择（占位按钮）
     - 球局类型选择（休闲 / 普通 / 强度）
     - 备注输入框
   - 底部大按钮：「确认发布」

4. **消息中心**
   - 约球状态列表（待成局 / 已满员 / 已过期）

5. **我的**
   - 用户头像 + 昵称（占位符）
   - 我发起的约球 / 我加入的约球列表
   - 设置按钮（退出登录）

## 输出要求
- 给出单个 HTML 文件，包含多个页面结构（用 `<section>` 分隔）。
- 每个页面顶部写注释标明页面名称。
- 样式简洁，确保在 iPhone 上可正常缩放和居中显示。
