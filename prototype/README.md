# 篮球约球小程序 - HTML原型

这是一个基于HTML + Tailwind CSS开发的篮球约球小程序原型，采用TDesign设计风格，展示了完整的用户界面和交互流程。

## 项目结构

```
prototype/
├── index.html      # 首页（地图模式）
├── venue.html      # 场馆详情页
├── create.html     # 发布约球页
└── README.md       # 项目说明文档
```

## 功能特性

### 1. 首页（index.html）
- **地图展示**：模拟腾讯地图SDK，显示用户位置和附近篮球场
- **场馆标记**：动态篮球场标记，点击可进入场馆详情
- **快速操作**：发布约球和搜索场馆的快捷按钮
- **附近约球**：显示附近的约球信息，支持一键加入
- **底部导航**：地图、场馆、发布、我的四个主要功能入口

### 2. 场馆详情页（venue.html）
- **场馆信息**：展示场馆名称、地址、开放时间、收费情况等
- **约球留言板**：类似聊天室的约球信息展示
- **约球卡片**：包含发布者信息、时间、人数、状态等
- **实时互动**：支持加入约球、查看参与人数
- **发布功能**：快速发布约球信息的入口

### 3. 发布约球页（create.html）
- **场馆选择**：预设场馆信息，支持更换
- **时间设置**：日期和时间段选择
- **人数配置**：半场3v3、全场5v5、自由练习等选项
- **技能要求**：新手友好、中等水平、高手切磋等标签
- **备注信息**：详细的约球要求描述
- **联系方式**：微信昵称和手机号码
- **其他设置**：提醒功能和公开设置

## 设计特点

### TDesign风格
- **色彩系统**：采用蓝色主色调，符合TDesign设计规范
- **组件设计**：卡片式布局，圆角设计，阴影效果
- **交互反馈**：悬停效果、点击状态、过渡动画
- **图标系统**：使用Heroicons风格的SVG图标

### 响应式设计
- **移动优先**：专为手机端设计，最大宽度限制在md断点
- **触摸友好**：按钮尺寸适中，支持触摸操作
- **视觉层次**：清晰的信息架构，易于浏览和操作

## 技术实现

### 前端技术
- **HTML5**：语义化标签，无障碍访问
- **Tailwind CSS**：原子化CSS框架，快速样式开发
- **JavaScript**：原生JS，实现交互功能
- **响应式设计**：移动端优先的布局设计

### 交互功能
- **页面导航**：支持页面间跳转和返回
- **表单交互**：技能标签选择、字数统计等
- **状态管理**：约球状态、加入状态等
- **模拟数据**：使用假数据展示完整功能

## 使用方法

### 本地预览
1. 直接在浏览器中打开任意HTML文件
2. 建议使用Chrome开发者工具的设备模拟器
3. 设置移动设备尺寸（如iPhone 12 Pro）

### 功能测试
1. **首页地图**：点击篮球场标记进入场馆详情
2. **场馆详情**：查看约球信息，点击加入按钮
3. **发布约球**：填写表单，选择各种选项
4. **页面导航**：使用底部导航栏切换功能

### 自定义修改
- 修改颜色主题：调整Tailwind CSS的颜色类
- 添加新功能：在现有页面基础上扩展
- 调整布局：修改CSS类和HTML结构

## 开发建议

### 微信小程序开发
1. **页面结构**：将HTML结构转换为WXML模板
2. **样式迁移**：将Tailwind类转换为WXSS样式
3. **交互逻辑**：使用微信小程序的JS API
4. **数据管理**：集成微信云开发数据库

### 云开发集成
1. **云函数**：实现约球发布、加入、查询等逻辑
2. **云数据库**：存储场馆信息、约球记录、用户数据
3. **云存储**：管理场馆图片、用户头像等资源
4. **订阅消息**：实现约球提醒功能
