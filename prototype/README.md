# 篮球约球 - HTML原型项目

## 项目概述

这是一个基于HTML + Tailwind CSS开发的"篮球约球"微信小程序原型项目，采用TDesign设计风格，专门适配iPhone15规格（393x852px）。

## 项目结构

```
prototype/
├── index.html          # 首页（地图模式）
├── venue.html          # 场馆详情页
├── create.html         # 发布约球页
├── mine.html           # 我的页面
└── README.md           # 项目说明文档
```

## 技术栈

- **HTML5**: 页面结构
- **Tailwind CSS**: 样式框架
- **TDesign Icons**: 图标库
- **原生JavaScript**: 交互逻辑

## 页面功能

### 首页 (index.html)
- **地图展示**: 模拟地图，显示用户位置和附近篮球场
- **搜索栏**: 顶部搜索球馆名称
- **场馆标记**: 可点击的篮球场标记，带动画效果
- **快速操作**: 发布约球、搜索场馆按钮
- **约球列表**: 显示附近正在进行的约球活动
- **底部导航**: 地图、场馆、发布、我的四个标签页

### 场馆详情页 (venue.html)
- **场馆信息**: 名称、地址、开放时间、收费标准等
- **约球留言板**: 类似聊天室的约球信息展示
- **实时更新**: 模拟30秒自动刷新时间显示
- **加入功能**: 可点击加入约球活动
- **发布入口**: 快速跳转到发布约球页面

### 发布约球页 (create.html)
- **场馆选择**: 支持多个场馆切换选择
- **时间选择**: 预设常用时间段，支持自定义
- **人数设置**: 下拉选择需要的人数
- **水平要求**: 新手友好、中等水平、高水平等标签
- **备注信息**: 200字限制的详细描述
- **其他设置**: 提醒开关、公开可见等选项

### 我的页面 (mine.html)
- **用户登录**: 模拟微信登录流程
- **个人信息**: 头像、昵称、手机号绑定
- **数据统计**: 发布次数、参与次数、信用评分
- **功能菜单**: 我的发布、我的参与、约球历史等
- **设置选项**: 消息提醒、意见反馈、关于我们
